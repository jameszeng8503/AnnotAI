AnnotAI Viewer
AnnotAI Viewer is a multilingual, browser-based viewer, annotator, presenter, and launcher for documents, media, URLs, AI tools, and social platforms.

It helps users open and review different types of content, draw directly on top of documents or media, present ideas visually, and quickly access popular English and Chinese AI tools and social media platforms from one convenient floating panel.

AnnotAI Viewer is designed as a lightweight web app that can run directly in the browser, making it suitable for teaching, research, meetings, content review, online collaboration, self-study, document presentation, media analysis, and AI-assisted workflows.

Overview
AnnotAI Viewer combines several useful functions into one browser-based workspace:
Document viewer
Media viewer
PDF annotator
Presentation assistant
Multilingual interface
URL loader
AI tool launcher
English and Chinese social platform launcher
Floating, draggable, resizable toolbar

Instead of switching between multiple apps, users can open files, paste URLs, annotate content, use a laser pointer, zoom in and out, save annotations, and launch AI or social platforms from the same interface.

Important Limitations
AnnotAI Viewer is powerful, but browser-based apps have natural limitations.

Local File Access
The app can only access local files selected by the user. It cannot freely browse your file system.

If the page is refreshed, local file object URLs may be lost, and the file may need to be opened again.

Website Embedding
Some websites block iframe embedding for security reasons.

If a webpage does not load inside the viewer, use Open External.

Cross-Origin Content
Some cross-origin images, videos, or embedded pages may not be exportable to PNG because of browser security restrictions.

Office and Special Formats
DOCX and spreadsheets are supported through browser-side conversion. However, complex formatting may not be perfectly preserved.

Formats like DOC, PPT, PPTX, RAR, and MOBI may require conversion or additional libraries for reliable preview.

Privacy and Security
AnnotAI Viewer is designed to work mostly inside the browser.

Local files are opened by user selection. The app does not automatically upload local files unless additional server-side features are added by the host or developer.

AI tools and social platforms opened from the launchers run in separate tabs and are governed by their own privacy policies, accounts, and terms of service.

Users should avoid uploading sensitive documents to external AI or social platforms unless they understand the privacy implications.
